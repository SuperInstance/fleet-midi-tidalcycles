# Student Guide to fleet-midi-tidalcycles

## What does this repo do?

Takes a list of -1s, 0s, and +1s (a ternary vector) and turns them into TidalCycles drum patterns. +1 = kick drum, 0 = hi-hat, -1 = snare.
